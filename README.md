# Playwright zpráva - dobrovolný úkol

- Vytvořte program za pomocí Playwright, který mi pošle zprávu (přes mail / teams / discord), ve kterém si zažádáte o jedničku.
- Bonus: Program se nejprve v konzoli zeptá na jméno vyučujícího, poté mu napíše email (ze jména automaticky odvodí email, který v bonusu nebude staticky nakódován a automaticky vytvoří oslovení). 

*Abychom se vyvarovali se spamování, první email je povolen, za každý další je strháván stupeň.*
