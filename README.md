# Playwright zpráva - dobrovolný úkol

- Vytvořte program za pomocí Playwright, který v Moodle "Úkol jednička" zašle nový příspěvek, ve kterém si zažádáte o jedničku.


**Důležité - neuploadujte svůj .env soubor!**
- kdo uploaduje z příkazového řádku, vytvořte v repozitáři nový soubor `.gitignore` (tečka na začátku je důležitá) a do něj napšite `*.env`
  - alternativně místo `git add .` / `git add -A` použijte `git add *název souboru.py*`
